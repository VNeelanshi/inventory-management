Global Filter System

Four Filters (FilterBar.vue)

Time Periodmonth → YYYY-MM or Qn-YYYY or "all"
Warehousewarehouse → San Francisco | London | Tokyo
Categorycategory → circuit boards | sensors | …
Order Statusstatus → delivered | shipped | processing | backordered

Stored as reactive refs in useFilters.js (singleton composable). All views share the same instance — changing a filter re-fetches data in every active view.

Filter Scope Per View

Dashboardwarehouse, category, status, month
Inventorywarehouse, category (no month — no time dimension)
Orderswarehouse, category, status, month
Spendingnone (spending endpoints take no filters)
Demandnone (/api/demand takes no filters)
Reportsnone (pre-aggregated quarterly/monthly data)

Backend skips a filter when its value is "all" or None. The frontend omits "all" params from the query string entirely.
